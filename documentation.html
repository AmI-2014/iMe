"The Treasure Hunting project is composed by two main systems: the first of these is the mobile application, realized in html5 with Ratchet and PhoneGap, which lets the players enjoy the game providing an interface for seeking checkpoints, solving clues and riddles, and checking the achievement of the treasure; the second part is the gameserver, written in python, using Django as web framework, and MySQL for the management of the database, which is in turn used to save usernames, lobbies, checkpoints and their properties and relations. Checkpoints in this first version of the application are located around the Politecnico di Torino."
